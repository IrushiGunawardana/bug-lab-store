# CTSE Buggy Shop (Standalone Demo Repo)

This is a standalone sample project for GitHub-context testing.

- It is intentionally simple and intentionally buggy.
- There are no `bugs/*.md` tickets in this repo.
- Your MAS can point to this repository using `--repo-owner` and `--repo-name`.
